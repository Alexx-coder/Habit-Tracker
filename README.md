# Tracker For Habits

Tracker For Habits - Трекер привычек, в которых надо вносить данные о выполнение привычки


## О Чем Tracker For Habits

- Tracker For Habits позволяет вам отслеживать и контролировать ваши привычки, получать достижения и тд

## Безопасность данных

- База данных: SQLite3

- Шифрование симметричным шифрованием (Fernet)

- Хэширование паролей используя Соль
